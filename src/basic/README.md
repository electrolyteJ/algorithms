  复杂度
============
表现形式：big 0 notation
时间复杂度的公式是： T(n) = O( f(n) )，其中f(n) 表示每行代码执行次数之和，而O 表示正比例关系
O(1)< O(log n) < O(n) < O(nlog n) < O(n^{2}) < O(2^{{n^{{\epsilon }}}}) < O(n!)
